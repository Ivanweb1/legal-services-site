# Прототип сайта юридических услуг (шаблон)

Прототип главной страницы для компании юридических/бизнес-услуг. Сделан как
переиспользуемый **шаблон**: единый файл без сборки, вся палитра и типографика
вынесены в CSS-переменные.

## Файлы

- `index.html` — прототип главной страницы. Самодостаточный (HTML + CSS + JS в одном файле), открывается двойным кликом.

## Структура главной

1. Шапка (sticky, навигация + телефон + CTA)
2. **Баннер** — 3 варианта
3. **Перечень услуг** — 3 варианта
4. Преимущества
5. **Блог** — 3 варианта
6. О компании
7. Форма заявки
8. **Сертификаты** — 3 варианта
9. Отзывы — место под готовый модуль Яндекс.Карт (+ превью верстки)
10. Контакты — карточка + место под карту
11. Футер

## Варианты блоков

У блоков с пометкой «3 вида» вверху есть тёмная панель с кнопками
**Вариант 1 / 2 / 3** — переключают оформление блока прямо на странице.
Панель — только инструмент прототипа, в продакшене остаётся один выбранный вариант.

| Блок | Вариант 1 | Вариант 2 | Вариант 3 |
|------|-----------|-----------|-----------|
| Баннер | Сплит: текст + форма заявки | Центрированный с крупным заголовком | Светлый минимализм + плитки услуг |
| Услуги | Карточки с иконкой и ценой | Нумерованный список в 2 колонки | Тайлы с одной выделенной |
| Блог | 3 равные карточки | Крупная статья + список | Горизонтальные строки |
| Сертификаты | Сетка «документов» | Лента наград (карусель) | Цифры доверия + миниатюры |

## Перекраска под другого клиента

Всё оформление — в блоке `:root` в начале `<style>`. Меняются в основном:

```css
--c-brand:  #14315c;   /* основной цвет бренда */
--c-accent: #c8a24a;   /* акцент (кнопки, подчёркивания) */
--font-head: Georgia, serif;   /* шрифт заголовков */
```

## Следующие шаги

- Выбрать финальные варианты блоков → убрать переключатель.
- Подставить реальный контент, фото, реквизиты клиента.
- Подключить модуль отзывов Яндекс.Карт и iframe карты в отмеченные места.
- Сверстать внутренние страницы: Услуги, Страница услуги, О компании, Контакты, Блог.
